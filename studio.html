Modular Synthesizer

Modular synthesizers are electronic musical instruments composed of separate modules that can be connected in various ways to create custom soundscapes. Unlike fixed-architecture synthesizers, modular synths allow users to design their own signal flow by patching cables between different units, offering nearly infinite possibilities for sound design.

I call this freedom.

Computer, DAW and Plugins

I mainly use the computer as a multitrack recorder. I have been using Logic Pro X as my main DAW for many years. I record via a RME Fireface 802, a Ferrorfish Pulse 16 and an UAD Apollo X4. I particularly like the compressors and equalisers from Universal Audio.

But no track lives without the Soundtoys Echoboy for delays. The RME analyser provides me with a very good visualrepresentation of the sound image.
